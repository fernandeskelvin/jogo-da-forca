## Jogo da Forca
No jogo da forca, os jogadores devem adivinhar uma palavra oculta com base na quantidade de letras e uma categoria temática. A cada erro, uma parte do boneco é desenhada, e o desafio é descobrir a palavra antes que a figura seja concluída.

## Editor de código
Visual Studio Code

## Sistema Operacional
Windows ou Linux
